# HistoryAttribute

#### Attribute
- 메타 데이터(코드에 대한 정보)를 담는 코드 요소
- 컴파일을 거치면 실행파일(어셈블리) 안에 저장되며, 컴퓨터가 런타임에 읽을 수 있음
- 이와는 달리, 주석은 사람만 읽을 수 있으며 컴파일을 거치면 실행파일에서는 제거됨
